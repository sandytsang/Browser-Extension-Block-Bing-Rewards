Installation Instructions (Linux Only)

  1. Download the main.crx file using the button above
  2. Open Chrome and navigate to chrome://extensions/
  3. Enable "Developer mode" using the toggle in the top-right corner
  4. Drag and drop the main.crx file onto the extensions page
  5. Click "Add extension" when prompted
  6. The extension will automatically check for updates

What it does

This extension blocks the URL https://www.bing.com/rewardsapp/widgetassets which is used by Bing to display the rewards icon and points counter.

Technical Details

  • Uses Manifest V3 (latest extension standard)
  • Uses declarativeNetRequest API for efficient URL blocking
  • No background scripts needed
  • Lightweight and efficient
