What it does

This extension blocks the URL https://www.bing.com/rewardsapp/widgetassets which is used by Bing to display the rewards icon and points counter.

Technical Details

  • Uses Manifest V3 (latest extension standard)
  • Uses declarativeNetRequest API for efficient URL blocking
  • No background scripts needed
  • Lightweight and efficient
